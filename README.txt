-> El código fuente esta en la carpeta src que tiene:
	- Main
	- Prac1 que es un package con nuestras clases

-> En la carpeta out/production/Practica-IA están los ficheros class compilados.

-> Para ejecutar el programa:

	java -classpath "direccion de \Practica-IA\out\production\Practica-IA; direccion de Practica-IA\Librerias\AIMA.jar; direccion de Practica-IA\Librerias\Gasolina.jar" Main

-> Ejemplo, en nuestro caso:

	java -classpath "C:\Users\David\Google Drive\UNI\IA\_Practica\Practica-IA\out\production\Practica-IA;C:\Users\David\Google Drive\UNI\IA\_Practica\Practica-IA\Librerias\AIMA.jar;C:\Users\David\Google Drive\UNI\IA\_Practica\Practica-IA\Librerias\Gasolina.jar" Main

-> En la carpeta raíz también hay dos archivos .jar en los cuales ya viene compilada la práctica con las librerias incluidas, para ejecturlo basta con hacer:

	java -jar IA_SA.jar

	java -jar IA_HC.jar
